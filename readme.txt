1. INTRODUCTION

This is a simple script that will take a list of mac addresses in a file, 1 per line, and change their format to a different format specified by the user.


2. REQUIREMENTS

This script was written in Python3.  The mac_format_module.py file has a function that will fetch the vendor of mac address.
This function is currently not used, it is written in for a future script idea.  But if you wish to use it, you will need to install the manuf module in Python3.
